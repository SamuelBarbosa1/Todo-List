For the installation of the project, follow the steps:

Clone the project:

```
https://github.com/SamuelBarbosa1/todo-list
```

Go to project path:

```
cd todo-list
```

Run the command to install all packages:

```
npm install
yarn install
```
Go to mongodb

```
package.json ""scripts": {start": "nodemon index.js",},"
cd api
yarn start
```
